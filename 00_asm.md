# base 10 (decimal)

```txt
eg: 13
    1 x (10^1) + 3 x (10^0)
    10 + 3
```

# base 2 (binary)

```
eg: 1010
    1 x (2^3) + 0 x (2^2) + 1 x (2^1) + 0 x (2^0)
    8 + 0 + 2 + 0 
    10
```
